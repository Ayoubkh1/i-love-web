Maandag 29-09-2025 Wat is het verschil tussen Flexbox Layout en Grid Layout? Flexbox gebruik je vooral voor één richting (rij of kolom), terwijl je met Grid meerdere rijen en kolommen tegelijk kunt indelen. -Wanneer zou je welke gebruiken? Flexbox gebruik ik als ik dingen horizontaal of verticaal wil uitlijnen, Grid als ik een complexere structuur wil met meerdere vlakken. -Wat is de default Layout Mode van elk element? De standaard layout is de flow layout. -Schrijf op welke Layout Modes je per onderdeel nodig hebt voor het onderwerp van de opdrachtgever Voor het plaatsen van artikelen gebruik ik Grid, en voor andere onderdelen zoals knoppen of navigatie gebruik ik Flexbox.
